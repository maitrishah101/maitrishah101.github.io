Awareable Technologies

In Spring 2019, I worked on my first Berkeley Innovation project with Awareable Technologies. It was really cool to work on a real world semester long project with an acutal client.

Some of my key contributions were research into competitor products, helping create a user reserach survey, interviewing a potential user, coming up with a number of key insights, designing some of the Mid-fi wireframes, and synthesizing user testing feedback. For the final product, I helped create the log in/sign up page and progress page. I also helped create some of the first iterations of the mascot.
